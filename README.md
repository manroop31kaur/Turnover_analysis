# Workforce Turnover Analysis

## Problem
Employee turnover increases hiring costs and impacts productivity. This project analyzes workforce data to identify turnover patterns and risk areas.

## Data
IBM HR Employee Attrition dataset (1,470 employees)

## Tools Used
- Power BI
- Excel
- DAX

## KPIs Created
- Turnover Rate
- Attrition by Department
- Attrition by Tenure

## Key Insights
- Turnover is highest within the first 1â€“2 years of employment
- Sales and Support roles show higher attrition
- Turnover decreases significantly with longer tenure

## Outcome
The analysis helps identify departments and tenure periods with higher retention risk to support workforce planning decisions.
